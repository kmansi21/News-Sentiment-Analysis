# News Article Sentiment Analysis

A web-based application built using **Python** and **Flask** that analyzes the sentiment of news articles.

---

## ðŸ“Œ Description

This project fetches real-time news articles using a **News API** and performs sentiment analysis on news titles using **TextBlob**.

It classifies news as **Positive**, **Negative**, or **Neutral** and displays sentiment distribution using a **pie chart**.  
It also supports **sentence-level sentiment analysis** for user-provided text.
